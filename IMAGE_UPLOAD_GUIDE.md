## 🎨 How It Works

### 1. User Uploads Image
```
User clicks upload area or drags image
    ↓
File validation (type & size)
    ↓
Upload to Supabase Storage (products bucket)
    ↓
Generate unique filename
    ↓
Get public URL
    ↓
Update form data with URL
    ↓
Show preview
```

### 2. Image Storage
- **Bucket**: `products`
- **Filename Format**: `{timestamp}-{random}.{extension}`
- **Example**: `1704123456789-abc123.jpg`
- **Public URL**: Automatically generated by Supabase

### 3. Database Storage
- URL saved to `image_url` field in `products` table
- No schema changes required

## 🔐 Supabase Storage Setup

### Required Bucket Configuration

1. **Bucket Name**: `products`
2. **Public Access**: Enabled
3. **File Size Limit**: 5MB (recommended)

### Create Bucket (if not exists)

```sql
-- In Supabase Dashboard → Storage → Create Bucket
-- Name: products
-- Public: Yes
```

### Bucket Policies

```sql
-- Allow authenticated users to upload
CREATE POLICY "Authenticated users can upload images"
ON storage.objects FOR INSERT
TO authenticated
WITH CHECK (bucket_id = 'products');

-- Allow public read access
CREATE POLICY "Public can view images"
ON storage.objects FOR SELECT
TO public
USING (bucket_id = 'products');

-- Allow authenticated users to delete their uploads
CREATE POLICY "Authenticated users can delete images"
ON storage.objects FOR DELETE
TO authenticated
USING (bucket_id = 'products');
```

## 🐛 Troubleshooting

### Issue: Upload Fails
**Possible Causes:**
- Bucket doesn't exist
- Bucket is not public
- User not authenticated
- File too large

**Solution:**
1. Check Supabase Storage dashboard
2. Verify bucket name is "products"
3. Ensure bucket is public
4. Check file size < 5MB

### Issue: Image Not Showing
**Possible Causes:**
- Invalid URL
- Bucket permissions
- CORS issues

**Solution:**
1. Check browser console for errors
2. Verify URL is accessible
3. Check Supabase bucket policies

### Issue: Slow Upload
**Possible Causes:**
- Large file size
- Slow internet connection
- Server issues

**Solution:**
1. Compress image before upload
2. Check internet connection
3. Try again later

## 📝 Code Examples

### Complete Form Integration

```typescript
"use client";

import { useState } from "react";
import ImageUpload from "@/app/components/ImageUpload";

export default function ProductForm() {
  const [formData, setFormData] = useState({
    title: "",
    image_url: "",
    price: "",
  });

  const handleSubmit = async (e: React.FormEvent) => {
    e.preventDefault();
    // Submit form with image_url
    console.log(formData);
  };

  return (
    <form onSubmit={handleSubmit}>
      <input
        type="text"
        value={formData.title}
        onChange={(e) => setFormData({ ...formData, title: e.target.value })}
        placeholder="Product Title"
      />

      <ImageUpload
        onUploadComplete={(url) => setFormData({ ...formData, image_url: url })}
        currentImageUrl={formData.image_url}
      />

      <input
        type="number"
        value={formData.price}
        onChange={(e) => setFormData({ ...formData, price: e.target.value })}
        placeholder="Price"
      />

      <button type="submit">Save Product</button>
    </form>
  );
}
```
